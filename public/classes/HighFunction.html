Class HighFunction

高次函数

处理N次函数定义，求值，方程求解问题

n次函数定义 f(x) = a0 * pow(x, n) + a1 * pow(x, n - 1) +.....+ an_1 * pow(x, 1) + an

0次 f(x) = a0; 1次 f(x) = a0 * x + a1; 2次 f(x) = a0 * x * x + a1 * x + a2; ......
